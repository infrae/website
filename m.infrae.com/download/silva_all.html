Silva Content Management System

Atom/RSS feed

Subscribe to a RSS or Atom feed of the Silva CMS Packages packages.

Silva is a powerful CMS for organizations that manage multiple or complex websites. Content is stored in clean and futureproof XML, independent of layout and presentation. Features include versioning, workflow system, integral visual editor (Kupu), content reuse, sophisticated access control, multi-site management, extensive import/export facilities, fine-grained templating, and hi-res image storage and manipulation. For complete information see the Silva product pages.

Product info

For complete information see the Silva product page.

User documentation is included in the product download. Recent changes can be found in the history file included with each release.

For a description of Silva dependencies and extensions, previous to the 2.1 release, see the Silva products map.

Installation

The recommended way to install Silva is using buildout to setup the software and all dependencies:

http://docs.infrae.com/silva/
