Silva External Sources

External Sources provide a facility in Silva to display information rendered by an external data source. The result can be easily inserted into Silva Documents.

External Sources are powerful tools, providing any functionality that’s possible with Zope, but not burdening an Author by requiring technical knowledge or syntax sensitive markup.

Included are:

  • a Code Source to include rendered page templates, forms, or results of Python code
  • a SQL Source to insert the result of a SQL query to an external database as a table
  • a CSV Source (a Silva asset with corresponding UI) to insert a CSV file as a table
