MOAI Server

PyPI

MOAI Server is also available on the Python Package Index:
http://pypi.python.org/pypi/MOAI

Product info

MOAI is a standalone server written in Python. There is a site for developer documentation at: http://moai.infrae.com/. For a general overview see the MOAI Server product page.

An Open Access Server Platform for Institutional Repositories

MOAI is a platform that aggregates input from disparate sources and republishes it via the Open Archive Initiatives protocol for metadata harvesting. MOAI is built for institutional repositories that store relational metadata and asset files.

The server has a flexible system for combining records into sets and uses these sets in the feed configuration. MOAI also comes with a simple yet flexible authentication scheme that can easily be customized. Besides providing authentication for the feeds, the authentication also controls access to the assets.

MOAI is a standalone system that can be used in combination with any repository software that comes with an OAI feed such as Fedora Commons, EPrints or DSpace. It can also be used directly with an SQL database or just a folder of XML files.
