Installation

Package information

  • Zope and Python versions for the 2.x series
  • Dependencies and Extensions for 2.1+
  • Products Map for 2.0
  • Products Map for 1.6
  • Products Map for 1.5
  • Products Map for 1.4

Since Silva 2.1, the recommended way to install Silva is using buildout to setup the software and all dependencies:
https://svn.infrae.com/buildout/silva/

For more explanation on using buildout, see Dependencies and Extensions for Silva and Installing Silva with Buildout.

Previous versions to 2.1 were installed by downloading a release tarball ‘Silva-all’ found below and unpacking it in an existing Zope instance Products directory.

For pre-2.0 versions there's also a ‘Silva-extra’ tarball available which contains optional but useful add-on and extension products.
